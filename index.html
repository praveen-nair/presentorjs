presentor.js
presentor.js is a modern presentation framework built using Javascript and CSS3 for the web. This is easy to use and can be customized to your needs.

Author: Praveen Nair

Adding a Slide using HTML
    <div id='presentor' width="100%" height="400px">
      <div id="slide1">
        <div id='slide1.title'>Slide 1</div>
        <div id='slide1.content'>Sample Content</div>
      </div>
      <div id="slide2">
        <div id='slide2.title'>Slide 1</div>
        <div id='slide2.content'>Sample Content</div>
      </div>
    </div>
Create a html div and ensure to give a unique ID. Place the slides in a div container. That it! Next initialize the presetor via javascript.

    <script>
    document.addEventListener( "DOMContentLoaded", function(event) {
      var ppt = window.presentor('presentor');
      ppt.setCyclicSideShow(true);
      ppt.addSlide("slide1");
      ppt.addSlide("slide2");
      ppt.initSlideShow('my_slide', 'fade');
      //Use setTimeout / Events to trigger the next slide to be shown.
      setTimeout(function() {
        ppt.gotoNextSlide('zoom_out_in');
      }, 4000); });
    </script>
Adding a Slide using Javascript
Want to dynamically add slide using JS? Here is how:


    <script>
    document.addEventListener( "DOMContentLoaded", function(event) {
      var ppt = window.presentor('presentor');
      ppt.createDynamicSlide({
        slideId: 'dyn_slide',
        slideTitle: 'This is a dynamic Slide 1',
        slideContent: 'The content for this slide is created Dynamically.' });
      ppt.initSlideShow('dyn_slide', 'fade');
    });
    </script>
Add Customized Slide
Add Customized slide in between by adding your own CSS Class to the Divs. Here's how:


    <div id="slide1" class="presentor_slide_custom">
      <div id='slide1.title' class="presentor_title_custom">Add Customized Slide</div>
      <div id='slide1.content' class="presentor_content_custom"> Add Customized slide in between by adding your own CSS Class to the Divs. </div>
    </div>
Want More Effects?
Not happy with the built-in effects? You can create your own effects and register it with presentor.js

    <script>
    document.addEventListener( "DOMContentLoaded", function(event) {
      var ppt = window.presentor('presentor');
      function newEffect(div_slideIdSrc, div_slideIdDest) {
        // Write your magical code to tranform div_slideIdSrc to div_slideIdDest
      }
      ppt.registerExternalEffects('new_effect', newEffect);
    });
    </script>
